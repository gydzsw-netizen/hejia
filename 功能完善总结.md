# 多国家运费核算功能完善总结

## 项目概述

本次更新为"光禹电商ERP系统 - 销售价格核算系统"添加了完整的多国家运费核算功能，使系统能够支持美国、德国、英国、法国、泛欧 5 个国家/地区的独立运费计算。

### 2. 数据库结构

#### 已有的迁移脚本
- `sql/add-country-settings.sql` - 44 个字段
- `sql/add-country-settings-part2.sql` - 11 个字段

**总计**：55 个新字段（5 个国家 × 11 个参数）

#### 字段列表（每个国家）
1. `{country}_profit_rate` - 利润率
2. `{country}_weight_rate` - 重量运费系数
3. `{country}_volume_factor` - 体积系数
4. `{country}_volume_rate` - 体积运费系数
5. `{country}_fixed_cost` - 固定成本
6. `{country}_min_price` - 最低售价
7. `{country}_usd_rate` - 美元汇率
8. `{country}_eur_rate` - 欧元汇率
9. `{country}_activity_rate` - 活动占比
10. `{country}_ad_rate` - 广告占比
11. `{country}_refund_rate` - 退款率

### 3. API 接口

#### 文件：`api/settings/index.js`
**状态**：已支持所有国家字段
- ALLOWED_FIELDS 包含所有 55 个国家字段
- GET 请求：读取用户设置
- POST 请求：保存用户设置

## 功能特性

### 用户功能
✅ 选择 5 个国家/地区
✅ 每个国家使用独立的运算规则
✅ 实时切换国家，自动加载对应规则
✅ 显示人民币、美元、欧元三种货币价格
✅ 详细的成本分解显示
✅ 国旗图标和国家名称显示

### 管理员功能
✅ 为每个国家独立设置运算规则
✅ 自动保存设置到数据库
✅ 设置活动占比和广告占比
✅ 不同国家的设置互不影响
✅ 实时预览设置效果
