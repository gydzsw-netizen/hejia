# 更新日志 (CHANGELOG)

## [2.0.0] - 2026-03-12

### 新增功能 ✨

#### 多国家运费核算
- 支持 5 个国家/地区的独立运费核算：
  - 🇺🇸 美国 (United States)
  - 🇩🇪 德国 (Germany)
  - 🇬🇧 英国 (United Kingdom)
  - 🇫🇷 法国 (France)
  - 🇪🇺 泛欧 (Pan-European)

- 每个国家独立的运算规则：
  - 利润率
  - 重量运费系数
  - 体积系数
  - 体积运费系数
  - 固定成本
  - 最低售价
  - 美元汇率
  - 欧元汇率
  - 活动占比
  - 广告占比
  - 退款率

#### 界面优化
- 优化国家选择器样式
  - 添加边框高亮效果
  - 添加悬停动画
  - 自定义下拉箭头
  - 添加国旗图标
- 在计算结果卡片中显示当前选择的国家
- 改进信息提示文本

### 改进 🔧

#### 功能改进
- 完善国家切换逻辑，确保活动占比和广告占比正确同步
- 优化设置保存机制，支持按国家保存
- 改进数据加载逻辑，优先使用国家特定设置

#### 代码优化
- 重构 `switchCountry()` 函数
- 优化 `applySettings()` 函数
- 改进 `getSettingsFromForm()` 函数
- 统一国家名称映射

### 数据库 💾

#### 新增字段
为 `user_settings` 表添加了 55 个新字段（每个国家 11 个字段）：
- 美国字段：`us_*`
- 德国字段：`de_*`
- 英国字段：`gb_*`
- 法国字段：`fr_*`
- 泛欧字段：`eu_*`

#### 迁移脚本
- `sql/add-country-settings.sql` - 添加美国、德国、英国、法国部分字段
- `sql/add-country-settings-part2.sql` - 添加法国剩余字段和泛欧字段

- `多国家运费核算使用说明.md` - 用户使用指南

## [1.0.0] - 2026-03-07

### 初始版本

#### 核心功能
- 用户认证系统（登录/登出）
- JWT Token 认证
- 管理员和普通用户权限区分
- 价格计算功能
- 用户管理功能（管理员）
- 运算规则配置（管理员）

#### 计算功能
- 基于重量和体积的运费计算
- 多货币显示（人民币、美元、欧元）
- 详细成本分解
- 利润率、活动占比、广告占比、退款率计算

#### 技术架构
- 前端：HTML5 + CSS3 + JavaScript (ES6 Modules)
- 后端：Vercel Serverless Functions
- 数据库：PostgreSQL (Vercel Postgres)
- 认证：JWT + bcrypt

#### 数据库表
- `users` - 用户表
- `user_settings` - 用户设置表
- `token_blacklist` - Token 黑名单表
