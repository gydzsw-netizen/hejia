# 多国家运费核算功能使用说明

## 功能概述

系统现已支持 5 个国家/地区的独立运费核算：
- 🇺🇸 **美国** (United States)
- 🇩🇪 **德国** (Germany)
- 🇬🇧 **英国** (United Kingdom)
- 🇫🇷 **法国** (France)
- 🇪🇺 **泛欧** (Pan-European)

每个国家都有独立的运算规则和参数设置。

## 使用方法

### 1. 选择目标国家

在价格核算页面顶部，点击"选择目标国家"下拉菜单，选择您要计算的目标市场。

### 2. 输入产品信息

- **产品重量**：以千克（kg）为单位
- **产品成本**：以人民币（元）为单位
- **产品尺寸**：长×宽×高，以厘米（cm）为单位

### 3. 查看运算规则（管理员）

管理员可以看到并修改每个国家的运算规则：
- **利润率**：目标利润百分比
- **活动占比**：平台活动费用占比
- **广告占比**：广告推广费用占比
- **退款率**：预估退款损失占比
- **重量运费系数**：每公斤的运费成本
- **体积系数**：用于计算体积重量（常用值：5000-6000）
- **体积运费系数**：体积重量的运费成本
- **固定成本**：包装、人工等固定成本
- **最低售价**：设置最低销售价格限制
- **美元汇率**：1 美元 = X 人民币
- **欧元汇率**：1 欧元 = X 人民币

### 4. 计算价格

点击"计算销售价格"按钮，系统会根据选择的国家使用对应的运算规则计算出：
- 人民币价格（CNY）
- 美元价格（USD）
- 欧元价格（EUR）
- 利润值
- 详细成本分解

## 计算公式

```
销售价格 = (产品成本 + 运费) / (1 - 利润率 - 活动占比 - 广告占比 - 退款率)

其中：
运费 = 重量运费 + 体积运费 + 固定成本
重量运费 = 实际重量 × 重量运费系数
体积运费 = 体积重量 × 体积运费系数
体积重量 = 长 × 宽 × 高 ÷ 体积系数
利润值 = 销售价格 - 产品成本 - 运费
```

## 管理员功能

### 设置不同国家的运算规则

1. 登录管理员账户
2. 进入价格核算页面
3. 选择目标国家
4. 在"运算规则"卡片中修改参数
5. 系统会自动保存修改

### 注意事项

- 每个国家的运算规则是独立的，互不影响
- 修改某个国家的规则不会影响其他国家
- 普通用户只能看到产品信息输入和计算结果，无法修改运算规则
- 活动占比和广告占比由管理员设置，普通用户不可修改

## 常见问题

**Q: 为什么不同国家的价格不一样？**
A: 不同国家的物流成本、市场环境、竞争情况都不同，因此需要使用不同的运算规则。

**Q: 如何设置合理的运算规则？**
A: 建议根据实际的物流成本、平台费用、市场竞争情况来设置。可以先使用默认值，然后根据实际销售情况逐步调整。

**Q: 普通用户可以修改运算规则吗？**
A: 不可以。只有管理员可以修改运算规则，普通用户只能使用管理员设置的规则进行计算。

**Q: 切换国家后，之前输入的产品信息会丢失吗？**
A: 不会。切换国家只会改变运算规则，产品信息（重量、成本、尺寸）会保留。
