# 销售价格核算系统

一个企业级的销售价格计算工具，支持多国家运费核算、多货币显示和自定义运算规则。

## 功能特点

- 🌍 **多国家运费核算**：支持美国、德国、英国、法国、泛欧 5 个国家/地区
- 📊 自动计算销售价格（基于成本、运费、利润率）
- 💱 多货币显示（人民币、美元、欧元）
- ⚙️ 可自定义运算规则（每个国家独立配置）
- 👥 用户认证系统（管理员和普通用户权限区分）
- 💾 数据持久化存储（PostgreSQL 数据库）
- 📱 响应式设计，支持手机和电脑访问

## 使用说明

### 多国家运费核算

系统支持 5 个国家/地区的独立运费核算：

| 国家/地区 | 代码 | 说明 |
|---------|------|------|
| 🇺🇸 美国 | US | United States |
| 🇩🇪 德国 | DE | Germany |
| 🇬🇧 英国 | GB | United Kingdom |
| 🇫🇷 法国 | FR | France |
| 🇪🇺 泛欧 | EU | Pan-European |

每个国家都有独立的运算规则配置，包括：
- 利润率、活动占比、广告占比、退款率
- 重量运费系数、体积系数、体积运费系数
- 固定成本、最低售价
- 美元汇率、欧元汇率

### 基本使用

1. 输入产品信息：
   - 产品重量（kg）
   - 产品成本（元）
   - 产品尺寸（长×宽×高，cm）

2. 点击"计算销售价格"按钮

3. 查看计算结果：
   - 人民币价格
   - 美元价格
   - 欧元价格
   - 详细成本分解

### 管理员设置

点击"管理员设置"按钮可以调整运算规则：

- **利润率**：在成本基础上增加的利润百分比
- **重量运费系数**：每公斤的运费成本
- **体积系数**：用于计算体积重量（常用值：5000-6000）
- **体积运费系数**：体积重量的运费成本
- **固定成本**：包装、人工等固定成本
- **最低售价**：设置最低销售价格限制
- **汇率设置**：美元和欧元的汇率

所有设置会自动保存到浏览器本地，下次访问时自动加载。

## 计算公式

```
销售价格 = (产品成本 + 重量运费 + 体积运费 + 固定成本) × (1 + 利润率)

其中：
- 重量运费 = 实际重量 × 重量运费系数
- 体积运费 = 体积重量 × 体积运费系数
- 体积重量 = 长 × 宽 × 高 ÷ 体积系数
```

## 常见问题

**Q: 员工的配置会互相影响吗？**
A: 不会。每个员工的配置保存在自己的浏览器中，互不影响。

**Q: 如何统一所有员工的运算规则？**
A: 可以在代码中修改默认值，然后重新部署。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）。

**Q: 可以离线使用吗？**
A: 首次访问后，浏览器会缓存页面，可以在离线状态下使用。
